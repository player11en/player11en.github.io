"Game Visuals"

Creating game based visuals for the event company Astro Agency.

The project involves creating visual games that can be used as a background for DJ events.
The visuals created are based on classic games such as Space Invaders, Street Fighter,Donkey Kong, Pacman and Snake.

Tools used for this project include Blender for 3D modeling, Affinity Photo for 2D design, Unity for programming and DaVinci Resolve for video editing.
