"Game Visuals"

Creating game based visuals for the event company Astro Agency.
The visuals are personalised for the events of Six Thirty (6:30), created to be entertaining a wide audience. The created videos are also working games.

The visuals are variation of:

Space Invader
Street Fighter
Donkey Kong
Pacman
Snake

Starting to make visually pleasing Animations is often all about keyframing and perfect setup. The setup makes it easy to render a perfect animation.

All games were created with Blender for 3D modelling,
Affinity Designer and Photo for image and texture generation,
Unity for coding and exporting the game as playables.
and finally Davinci Resolve for video cutting and rendering.
