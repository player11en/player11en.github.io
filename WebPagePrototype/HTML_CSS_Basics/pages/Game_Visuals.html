"Game Visuals"

Erstellen von Spiel-basierten Visuals für die Eventagentur Astro Agency.

Das Projekt umfasst die Erstellung von visuellen Spielen, die als Hintergrund für DJ-Events verwendet werden können.
Die erstellten Visuals basieren auf klassischen Spielen wie Space Invaders, Street Fighter, Donkey Kong, Pacman und Snake.


Für das Projekt wurde verwendet
  • Blender (3D Modelling/Animation)
  • Affinity Photo/Designer (2D Design)
  • Unity (Programming)
  • DaVinci Resolve (Video editing)
