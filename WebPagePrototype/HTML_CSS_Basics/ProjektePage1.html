"Park Memories -- A visual storytelling experince"

"Park Memories -- A visual storytelling experince"
is a small webbased storytelling narrative. The WebApplication tells a story about an incident that happend in the park a long time ago. Through notes and other clues the story of the park and the incident is told. The user should be motivated to explore the area.
The narrative is still in develeopment. This is a prototypical concept where the key is to test features and create a bigger and more complex world where the user can experience a full story with more interactions and decisions.

Park Memories -- A visual storytelling experince

For "Park Memories" we have used the WebGL framework Babylon.js. Babylon.js is a powerful web rendering engine that is developed by microsoft.
it has powerful features and a clean and orgaized documentation. Babylon.js is fully Javascript based.
The park was created with Blender, before using the model it was been cleaned up and the mesh and polygon count has also been reduced. Using different postprocessing and a self written walking and sprinting script helps the user to freely move and enjoy the small atmosphere
